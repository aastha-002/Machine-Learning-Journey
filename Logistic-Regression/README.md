# Logistic Regression

## Dataset
Heart Disease Dataset (heart.csv)

## Libraries Used
- Pandas
- Scikit-learn

## Steps
1. Import libraries
2. Load dataset
3. Split data into training and testing sets
4. Train Logistic Regression model
5. Predict on test data
6. Calculate Accuracy and Precision

## Results
Accuracy: 88.52%
Precision: 87.88%
