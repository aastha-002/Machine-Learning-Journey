# Naive Bayes Classification

## Project Overview
This project demonstrates the implementation of the Gaussian Naive Bayes algorithm using Scikit-learn to predict heart disease.

## Dataset
- heart.csv

## Libraries Used
- Pandas
- Scikit-learn

## Steps Performed
1. Imported required libraries.
2. Loaded the dataset.
3. Split data into training and testing sets.
4. Trained Gaussian Naive Bayes model.
5. Predicted test data.
6. Evaluated the model.

## Evaluation Metrics
- Accuracy Score
- Precision Score
- Recall Score

## Results
- Accuracy: **86.89%**
- Precision: **90%**
- Recall: **84.38%**
