# Linear Regression - Medical Insurance Cost Prediction

## Overview 
This project uses Linear Regression to predict medical Insurance charges based on factors such as age,gender,BMI,no.of children and smoking status.

## Dataset
insurance.csv

## Libraries Used
- Pandas
- Seaborn
- Scikit-Learn

## Steps Performed
1. Data Loading
2. Data Visualization
3. Data Preprocessing
4. Feature Encoding
5. Train-Test Split
6. Linear Regression Model Training 
7. Prediction
8. Model Evaluation

## Result
R² Score: 0.7811
